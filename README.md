# 2nd-year
This repository contains codes and pre-trained checkpoints of a place recognition model for Friends video.  

# Requirements
- Ubuntu (tested on v14.04)
- Python (tested on v2.7)
- Pytorch (tested on v0.4.1)
- Numpy
- PIL
- opencv-python
- matplotlib
